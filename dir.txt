DESCRIPTION OF DIRECTORIES:

build	    - where object files are placed
dev	    - development files, mostly licences
images	    - where object files are placed
include	    - stuff included in source
src	    - the meat of CoCoBoot2 source
tools	    - compile-time tools required to build


DESCRIPTION OF TOOLS DIRECTORY:

bfc	    - Brett's Forth Compiler a Cross-Compiler for forth source
idiff	    - An HDBDOS image compresser
rofs	    - Read Only File System util for building a .dsk file
slz	    - SLZ compressor - for compressing the boot-track
